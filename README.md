# 文字起こしツール

Apple Silicon Mac 向けの、日本語動画・音声の文字起こし & 話者分離デスクトップアプリ。

- **mlx-whisper** (Apple MLX) による高速文字起こし ― Metal GPU をフル活用
- **pyannote.audio** による話者分離 ― 誰が何を話したかを自動判別
- 動画・音声ファイルを **ドラッグ＆ドロップ** するだけ
- 複数ファイルを **バッチ処理** 可能
- すべて **ローカル処理**（音声・テキストは外部に送信されません）

## 動作環境

| 項目 | 必要条件 |
|---|---|
| OS | macOS（Apple Silicon 必須 / M1 以降）|
| メモリ | **8GB 以上**（8〜16GB は「軽量モード」推奨、24GB 以上は標準モードでフル活用）|
| Python | 3.10 以上（`install.sh` が自動でセットアップ）|
| 通信 | 初回モデルダウンロード時のみ必要 |

> 💡 **メモリ判定は自動**: 起動時に `psutil` が物理メモリを検出し、16GB 以下なら「軽量モード」と推奨モデル `small` を自動で設定します。
