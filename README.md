# 文字起こしツール

Faster Whisper + 話者分離（pyannote）による日本語文字起こしツール。

- 動画・音声ファイルを投げるだけで文字起こし
- 誰が話しているか自動判別（話者分離）
- 複数ファイルをまとめて処理可能（バッチ対応）
- Mac ネイティブアプリとして動作（ローカル処理・データ外部送信なし）

## セットアップ

→ **[セットアップ手順書](docs/セットアップ手順書.md)** を参照してください。

## 動作環境

- macOS（Apple Silicon 推奨）
- Python 3.10 以上
- インターネット接続（初回モデルダウンロード時のみ）
